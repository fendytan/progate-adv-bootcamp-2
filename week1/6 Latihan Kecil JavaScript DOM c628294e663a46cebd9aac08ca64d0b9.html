1. Program: Horoskop

Program horoskop ini menghasilkan nilai acak setiap kali Anda menjalankan program tersebut.

Seperti yang Anda dapat perhatikan dalam gambar di bawah ini, program menampilkan apakah Anda sedang beruntung atau tidak hari ini.

Program dapat menghasilkan salah satu di antara Hebat!! , Boleh juga! atau Mengerikan....

Menggunakan pengetahuan memanipulasi DOM yang telah Anda pelajari, mari kita coba untuk menghasilkan nilai acak setiap Anda klik tombolnya!

Petunjuk:

Ide dasar dari program ini adalah:

  1. Pertama-tama, Anda hanya perlu mencetak Keberuntungan Anda hari ini adalah: sebagai sebuah string.
  1. Untuk menghasilkan sebuah nilai yang acak, Anda perlu menghasilkan sebuah angka acak. Dapat diwujudkan dengan code ini:
const randomNumber = Date.now();

Karena code di atas tersebut menghasilkan waktu saat ini dalam bentuk seperseribu detik sejak 1 January 1970 UTC, nilai yang dihasilkan pun berubah setiap kali Anda menjalankan code tersebut.

  1. Melalui code di atas yang menghasilkan sebuah angka seperti 1631502977737 , Anda hanya memerlukan 3 pola untuk menampilkan 3 hasil yang berbeda (Hebat!! , Boleh juga! atau Mengerikan...). Mari ubah randomNumber sehingga menghasilkan salah satu diantara1, 2 atau 3 dengan melakukan sebuah perhitungan.
  1. Terakhir, tampilkan hasil yang tergantung pada angka yang Anda terima dari proses sebelumnya.

Latihan Tambahan:

  • Cobalah menggunakan pernyataan switch ketika hendak menggunakan pernyataan if dan sebaliknya.
  • Cobalah menambahkan hasilnya misalnya menjadi 5 pola atau lebih dan perhatikan mana yang lebih mudah untuk Anda, apakah switch atau if.
  • Cobalah untuk mengubah peluangnya, misalnya seperti Hebat!!: 20% , Boleh juga!: 60% atau Mengerikan...: 20%.

2. Program: FizzBuzz

Mari kita uji kemampuan DOM Anda lagi!

Berikut bagaimana program ini bekerja:

  1. Siapkan sebuah formulir untuk input sebuah angka.
  1. Tampilkan hasil angka FizzBuzz di formulir tersebut saat tombol di klik.

(Seperti yang ditampilkan di bawah, saat angka 100 diberikan sebagai input pada formulir, Buzz! muncul sesuai yang diharapkan, dan begitu juga halnya jika angka selain 89 yang dilakukan input pada formulir ini.)

  1. Namun, nilai invalid input akan muncul jika formulir kosong atau jika inputnya adalah angka 0.
