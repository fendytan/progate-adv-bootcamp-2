Asynchronous Javascript and API

We will use asynchronous process when asking data from another server. Asynchronous will make our code more efficient since we can do more code execution and still running process that usually takes longer time to finished.

Show Pokemon on Gameboy

Show Pikachu data using 2 endpoints:
- https://pokeapi.co/api/v2/pokemon/pikachu
- https://pokeapi.co/api/v2/pokemon/25

Needed data to show are number, name, image and type.

  • Showing the Pokemon data on Gameboy CSS with interactive buttons.
  • Press button A to show Pikachu data.
  • Press button B for random Pokemon (only the first 151 original Pokemon).
  • Press LEFT or RIGHT will show previous/next Pokemon.
  • User can enter a valid Pokemon ID or name to show specific Pokemon.
    This to show that this program can receive data using both endpoints.
